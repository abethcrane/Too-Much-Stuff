Book Library

Solving the problem of which book to buy for a bibliophile.

Download this project as a .zip file Download this project as a tar.gz file

The Premise

Buying books is hard - what if the person already owns it, or you don't know what their taste is like? Simple - check out the list of all books they own in an easy to use app or site.

Accounts

To get set up just make an account - easy as! Then scan a barcode or enter an ISBN, and it'll add it to your lists. You can search users via email address, and once you're on a page just type in an author/title to search their list.

Languages

The Android App uses the Java SDK, the website front-end is basic HTML/CSS/Javascript, and the back end is a MySQL database.

Authors and Contributors

@bethc struggles to buy @AvengingSyndrome books he doesn't already own. So instead of just not buying him books, they built an easy way of checking which books he has.
